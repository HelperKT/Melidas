Privacy Policy

Effective Date: June 15, 2026

This Privacy Policy explains how this Discord Bot collects, stores and uses information.

1. Information Collected

The Bot may collect and store:

  • Discord User IDs
  • Discord Server IDs
  • User XP (experience points)
  • User levels
  • Message counts
  • Profile theme preferences

2. Information Not Collected

The Bot does not collect:

  • Passwords
  • Email addresses
  • Payment information
  • IP addresses
  • Message contents

3. How Data Is Used

Collected data is used solely to:

  • Calculate XP and levels
  • Generate rank cards
  • Display rankings and leaderboards
  • Store profile settings
  • Provide Bot functionality

4. Data Sharing

User data is not sold, rented, or shared with third parties.

5. Data Storage

Data is stored only as long as necessary for operation of the Bot.

6. Data Deletion

Users may request deletion of their data by contacting the Bot owner.

7. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the Bot constitutes acceptance of the updated policy.
